# Project Summary
**NXT** is a digital wallet application designed for seamless cryptocurrency management and financial transactions on both mobile and desktop platforms. It features innovative functionalities such as "Transfer by Location," "Transfer via QR," budgeting tools, and analytics, empowering users with enhanced financial oversight. NXT emphasizes user experience, security, and modern aesthetics, making it a vital tool for effective digital finance management. Recent updates include mobile-specific video support for enhanced marketing engagement.

# Project Module Description
NXT includes the following functional modules:
- **Responsive Design**: Adapts seamlessly to various devices.
- **Wallet Features**: Users can track balances and perform transactions.
- **Custom Navigation**: Bottom sheet menu with swipe-to-close functionality.
- **UI Components**: Utilizes Shadcn UI for a contemporary look.
- **Statistics Section**: Offers personalized financial insights.
- **Fintech-Level Security**: Implements stringent user protection measures.
- **Dynamic Backgrounds**: Enhances user experience with smooth transitions and animations.
- **Pricing Section**: Subscription plans: **XZERO** (Free), **XPRESS** ($2.00/month), **XCLUSIVE** ($12.00/month).
- **Marketing Sections**: Engaging sticky video banners for promotional content, now includes mobile-specific videos.
- **Legal Pages**: Includes Rights and Duties, Terms and Conditions, and Privacy and Cookies pages.
- **Animations**: Boosts user engagement through interactive elements.
- **Cookie Consent Banner**: A compact and functional consent banner.
- **Enhanced Feature Images**: Improves visibility in the features section.
- **Pricing Separator**: Thin border line added between pricing and features sections for better visual separation.
- **Re-triggerable Animations**: Allows animations to reset and replay when elements enter the viewport.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Typed superset of JavaScript for safer code.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for fast development and optimized builds.
- **Shadcn UI**: Component library for modern UI elements.
- **React Router**: Library for routing and navigation within the application.

# Usage
To set up and run the project:
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run start
   ```
4. For development, run:
   ```bash
   pnpm run dev
   ```
